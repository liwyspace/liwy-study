3. 补充知识

3.1 IE兼容性问题

  • Feature Internet    Explorer 8     Internet Explorer 9
border-radius       不支持         支持
box-shadow          不支持         支持
transform           不支持         支持, with -ms prefix
transition          不支持
placeholder         不支持
  • Internet Explorer 8 需要 Respond.js 配合才能实现对媒体查询（media query）的支持。
  • Respond.js 不支持通过 @import 指令所引入的 CSS 文件。例如，Drupal 一般被配置为通过 @import 指令引入CSS文件，Respond.js 对其将无法起到作用。
  • 当 box-sizing: border-box; 与 min-width、max-width、min-height 或 max-height 一同使用时，IE8 不能完全支持 box-sizing 属性。
  • 当 @font-face 与 :before 在 IE8 下共同使用时会出现问题。
  • Bootstrap 不支持 IE 古老的兼容模式。为了让 IE 浏览器运行最新的渲染模式下，建议将此 标签加入到你的页面中： 
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
  • 将下面的 标签加入到页面中，可以让部分国产浏览器默认采用高速模式渲染页面： 
    <meta name="renderer" content="webkit">
