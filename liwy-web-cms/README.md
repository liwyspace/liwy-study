# 命令备忘
## 项目初始化
```bash
# 初始化npm
npm init
# 初始化bower
bower init
# 安装gulp
npm install --save-dev gulp
# 安装jquery
bower install --save jquery
# 安装bootstrap
bower install --save bootstrap

```

## git分支操作
```bash
# 查看分支清单
git branch
# 创建分支
git branch testing
# 切换到分支
git checkout testing
# 新建并切换分支
git checkout -b testing
# 合并分支到当前分支
git merge testing
# 删除分支
git branch -d testing
```
